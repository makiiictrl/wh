Main Navigation

We have many navigation modes, basically split into two categories: mini and max ( including all off-canvas modes). To control the submenu collapse mini-navigation floating menu, we need a plugin called MainNavigation.

When you are not using main navigation or just regular links without submenus, and you are not using mini-navigation, you can skip this plugin from your project.

Default variables

MainNavigation come with default variables, but sometimes those don't match what you need, so you just need to change them for the entire element or specific component.

Name Type Default Description
eventSidebarsChange string change.nf.sidebars The default event type will be fired when the sidebar mode is changed.
eventOutsideTrigger array [ "click", "touchend" ] The mini-navigation event type that tells any active floating submenu should be hidden.
eventSubmenuToggler array [ "click", "mouseenter" ] This is the event type used to toggle the submenu or a floating submenu.
selectorRoot string #root The class name or id of the root element.
selectorContent string #content The class name or id of the content element.
selectorNav string #mainnav-container The class name or id of the navigation element.
selectorMininavToggle string .mininav-toggle The class names of the submenu togglers.
selectorMininavContent string .mininav-content The class names of submenus content.

Get the default variables

nifty.MainNavigation.Default

Modify the default variable.

nifty.MainNavigation.Default.selectorNav = "#my-navigation"
nifty.MainNavigation.Default.selectorContent = "#my-content-container"
...

Methods

Method Description
mode Static method, which allows you to get the current navigation mode. Please check out the example code below.
setmode Static method, allows you to set the navigation to a new mode. Please check out the example code below.
update Update the MainNavigation component. Include updating all selectors, updating the navigation mode, and initializing the collapse submenu based on the navigation mode.

Get the current navigation mode.

nifty.MainNavigation.mode
// It will return a value: "maxi", "mini", "push", "slide", or "reveal".

Set the navigation mode.

nifty.MainNavigation.setmode("mini")
// The mode value should be something like: "maxi", "mini", "push", "slide", or "reveal".
