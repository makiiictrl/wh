Sidebar Toggler

Turn your anchors or buttons into a sidebar toggle. You can toggle the navigation modes, or show and hide the right side sidebar.

Usage

<button class="btn btn-primary nav-toggler">Toggle Navigation</button>

<button class="btn btn-primary sidebar-toggler">Toggle Sidebar</button>

Default variables

It comes with default variables but allows for modifying variables on the page load upon initialization.

Name Type Default Description
event string click The default event type to toggle the sidebars.
eventSidebarsChange string change.nf.sidebar The event name is fired immediately when the toggle method has been called.
eventSidebarsChanged string changed.nf.sidebar The event name is fired when the toggle method and CSS transition are completed.
selectorRoot string #root The class name or id of the root element.
selectorToggleNav string .nav-toggler A class name for the navigation toggler button.
selectorToggleSidebar string .sidebar-toggler A class name for the right side sidebar toggler button.
selectorNav string .mainnav The id or class name for the navigation element.
selectorSidebar string .sidebar The id or class name for the sidebar element.

Get the default variables

nifty.SidebarToggler.Default

Modify the default variable.

nifty.SidebarToggler.Default.selectorNav = "#my-navigation"
nifty.SidebarToggler.Default.selectorSidebar = "#my-sidebar"
...

Event

The Switcher events are fired at the toggling element and then bubbled up. You can also add event listeners to the document or the element.

Event type Description
change.nf.sidebar Fired immediately when the toggle method has been called.
changed.nf.sidebar Fired when the toggle method and CSS transition are completed. 
document.addEventListener( "change.nf.sidebar", event => {
  // do something...
})

// or

const myNavigation = document.querySelector( "#mainnav-container" )
myNavigation.addEventListener( "changed.nf.sidebar", event => {
  // do something...
})
