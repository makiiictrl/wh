Build Tools

We're including three build tools to save you time once you've gotten started with the theme. All the build tools have the same command line to get started with development and build your app for production.

Here is the list of build tools that are included in the Getting Started folder.

Gulp gulpjs.com
Leverage gulp and the flexibility of JavaScript to automate slow, repetitive workflows and compose them into efficient build pipelines.
Parcel parceljs.org
Parcel combines a great out-of-the-box development experience with a scalable architecture that can take your project from just getting started to massive production application.
Vite vitejs.dev
Vite is the next generation frontend tooling, Get ready for a development environment that can finally catch up with you.

Getting Started

Please make sure you have NodeJS and ( npm, yarn, or pnpm ) installed on your machine.

  • First install dependencies:

    npm install

    yarn

    pnpm install

  • To run in hot module reloading mode:

    npm run dev

    yarn dev

    pnpm run dev

    Then go to: http://localhost:3000

  • To create a production build:

    npm run build

    yarn build

    pnpm run build
