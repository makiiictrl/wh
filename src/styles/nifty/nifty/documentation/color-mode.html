Color Mode

There are two color mode build-ins: dark and light. You can switch between color modes by adding the attribute data-bs-theme="light" or data-bs-theme="dark" in the HTML tag at the top of the document.

<!DOCTYPE html>
<html lang="en" data-bs-theme="light">
   <head>
      ...
      ...

For the simplest of methods, just install our plugin called Color-Mode-Switcher. The plugin can be applied to checkboxes, radio buttons, or select components.
