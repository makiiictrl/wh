Toggler Buttons

Use this plugin to create a helpful button to display your content. You can make a fullscreen browser, full-page browser content or toggle a class for some element.

Fullscreen & Full-page

<button class="btn btn-primary btn-lg" data-nf-toggler="fullscreen" data-nf-target="html">Fullscreen</button>

Targeting an element

Make fullscreen by targeting the element; in this case, we target the card element as fullscreen or just a full-page content browser.

Default variables

It comes with default variables but allows for modifying variables on the page load upon initialization.

Name Type Default Description
event string click The default event type for applying an action to a target.
dataKey string data-nf-toggler Name of the attribute to indicate the toggler button.
dataTargetKey string data-nf-target Name the attribute that will add a target selector when the button is clicked.
dataClassKey string data-nf-class This attribute is only available when your button targets an element to toggle a class. It should be filled with a class name.
toggleMode object {
 "FullScreen" : "fullscreen",  "FullPage" : "fullpage",  "Dismiss" : "dismiss",  "Class" : "class"
} 		List of functions that are currently supported. You might need it to change the object's value to your language.

Get the default variables

nifty.Toggler.Default

Modify the default variable.

nifty.Toggler.Default.dataKey = "data-my-toggler"
nifty.Toggler.Default.dataTargetKey = "data-my-target"
...
