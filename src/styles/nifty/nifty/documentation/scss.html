SCSS Files

We build the theme by overriding most of Bootstrap's variables and then recompile it to keep the file size small.

If you have a custom style or just want to override some of Nifty's variables, please add them to the _styles.scss file. You can find the file at scss/my-custom-styles/_styles.scss

We've also added a few seed projects and build tools like Gulp, Parcel, and Vite. It will be easier to override the SCSS variables and add custom styles.

If you make changes directly in the CSS and SCSS files from Nifty, it will be hard to update them once we release an update. For this purpose, we recommend creating a new CSS/ SCSS file.
