Color Mode Switcher

Turn your checkboxes, radio buttons, or select components into the Color mode switcher. Add a .mode-switcher class that will automatically initialize and turn it into a toggler component.

Select

These select components have the Auto option.

The Auto option is removed from this component.

<select class="form-select mode-switcher" aria-label="select color mode" autocomplete="off">
   <option value="auto">Auto</option> <!-- With or without the Auto option -->
   <option value="light">Light Mode</option>
   <option value="dark">Dark Mode</option>
</select>

Checkbox

This is a minimal mode with no added value attributes.

Add a value light and dark to the checkbox but without an auto checkbox

All checkboxes have values; add an auto checkbox to the list.

<!-- This is a minimal mode with no added value attributes. -->

   <label for="_dm-cbBasicColorMode" class="form-check-label">Dark Mode</label>
</div>

Switches

A switch has the markup of a custom checkbox but uses the .form-switch class to render a toggle switch.

<!-- Basic example -->
<div class="form-check form-switch">
   <input id="_dm-cbSwitchesColorMode_1" class="form-check-input mode-switcher" type="checkbox" autocomplete="off">
   <label for="_dm-cbSwitchesColorMode_1" class="form-check-label ms-2">Dark Mode</label>
</div>

<!-- Swhitch with icons -->
<div class="form-check form-switch">
   <input id="_dm-cbSwitchesIconsColorMode_1" class="form-check-input mode-switcher" type="checkbox" role="switch">
   <label for="_dm-cbSwitchesIconsColorMode_1" class="form-check-label ps-1 fw-bold d-none d-md-flex align-items-center ">
      <span class="mode-switcher-icon icon-light ms-2 text-primary">
         <!-- Add your icon here -->
         <span class="ms-1 fw-normal">Light Mode</span>
      </span>
      <span class="mode-switcher-icon icon-dark ms-2">
         <!-- Add your icon here -->
         <span class="ms-1 fw-normal">Dark Mode</span>
      </span>
   </label>
</div>

Usage

Call the switcher via JavaScript

const colorModeList = document.querySelectorAll( ".mode-switcher" )
const elementList = [...colorModeList].map( switcher => {
   new nifty.ColorModeSwitcher( switcher )
})

Default variables

Switchers come with default variables, but sometimes those don't match what you need, so you just need to change them for the entire element or specific component.

Name Type Default Description
event string change The default event type will be fired when the component makes changes.
selectorSwitcher string .mode-switcher Default class name for the switcher.
selectorSwitcherIcons string .mode-switcher-icon Icon class name used for toggling show/hide the icon when color mode changes.
dataBSColorMode string data-bs-theme Default attribute provided by Bootstrap to indicate the currently selected color mode
(you may need this sometime when the attribute is changed)
availableColorMode array [ "auto", "light", "dark" ] Available color modes

Get the default variables

nifty.ColorModeSwitcher.Default

Modify the default variable for the entire switch.

nifty.ColorModeSwitcher.Default.selectorSwitcher = ".color-mode-toggler"
nifty.ColorModeSwitcher.Default.selectorSwitcherIcons = ".color-mode-toggler-icon"
nifty.ColorModeSwitcher.Default.dataBSColorMode = "bs-theme-mode"
...

For the specific component.

const mySwitcher = new nifty.ColorModeSwitcher( document.querySelector( ".color-mode-toggler" ), {
   selectorSwitcherIcons: ".color-mode-toggler-icon",
   dataBSColorMode: "bs-color-mode",
   ...
})

Methods

Method Description
dispose Destroys an element’s switcher. (Removes stored data on the DOM element)
setColorMode Set the color mode to a new value. This method is also available in the static method, please check out the example code below.
prefersTheme Static method which allows you to get the user's preferred theme. Please check out the example code below.
update Updates the state of an element’s switcher.

Set the Color Mode to auto, light, or dark.

nifty.ColorModeSwitcher.setColorMode( "auto" ) // "auto", "light", or "dark"

Get the user's preferred theme

nifty.ColorModeSwitcher.prefersTheme
// It will return a value: "light" or "dark" based on user system settings

Event

The Switcher events are fired at the toggling element and then bubbled up. You can also add event listeners to the document.

Event type Description
change.nf.colormode Fired when the switcher was changed.
