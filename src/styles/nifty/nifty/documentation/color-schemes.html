Color Schemes

We have included 12 color schemes to match your requirements. All color schemes are included in the single compiled CSS, and there is no need to load any external CSS to apply a different color scheme.

Modify the colors

You can easily edit, delete the colors, or even add more color schemes by overriding the following variables as references in the file: scss/nifty-admin/colors/schemes/_default.scss.

Please keep in mind that each color scheme will automatically generate its components and the element's colors. So, to create a new color scheme, you just need to define one primary color.

// COLOR SCHEMES
// ------------------------------------------------------
$color-schemes: (
   "gray":    #445169,
   "navy":    $indigo,
   "ocean":   #0057ad,
   "lime":    #3f911d,
   "violet":  #6c307e,
   "orange":  #ff530f,
   "teal":    #037c7c,
   "corn":    #e3cb0d,
   "cherry":  #e0383e,
   "coffee":  #74543d,
   "pear":    #852949,
   "night":   #24282f
) !default;

Applying the color scheme

Applying the color scheme on the page is simple by adding the attribute data-scheme="color-name" to the HTML tag on top, which is the default color scheme for the page.

<!DOCTYPE html>
<html lang="en" data-scheme="navy">
   <head>
      ...
      ...
